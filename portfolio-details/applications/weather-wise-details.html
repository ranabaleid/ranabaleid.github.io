WeatherWise


Project information

  • Category: Application
  • Demo URL: Figma Demo
  • User Interface URL: Figma UI
  • Presentation URL: Portfolio
  • Description: The WeatherWise app provides personalized activity recommendations based on real-time weather conditions, helping users plan their daily experiences more effectively. By offering location-based suggestions, such as outdoor activities on sunny days or indoor options during rain, the app ensures users make the most of their time. Features include weather forecasts, nearby activity suggestions with descriptions and images, and user-friendly settings for location and notification preferences. Designed in Figma, the app prioritizes simplicity, accessibility, and adaptability, creating a seamless experience for all users.
