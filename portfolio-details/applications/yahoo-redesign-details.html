Yahoo! Redesign

Project information

  • Category: Application
  • Demo URL: Figma Link
  • Description A redesign of Yahoo's interface aimed to improve usability and user experience. The study utilized Think Aloud Testing, Quantitative Analysis, and Heuristic Evaluation to gather feedback from users and experts. Improvements focused on enhancing visual flow, consistency, and ease of navigation, resulting in a more cohesive and user-friendly interface. The final design was developed using Figma, addressing user preferences and making Yahoo easier and more intuitive to use.
